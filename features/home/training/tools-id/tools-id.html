ابزار شناسی (Tools Identification)

ابزارهای خط تولید و کنترل کیفیت

استفاده از ابزار مناسب و کالیبره، تضمین کننده صحت فرآیند تولید و کنترل است.

۱. ابزارهای اندازه‌گیری

  • **کولیس دیجیتال:** برای اندازه‌گیری دقیق ابعاد قطعات پلاستیکی و فلزی.
  • **مولتی‌متر:** برای تست مدارهای الکتریکی، ولتاژ و مقاومت.
  • **تراز لیزری:** برای تنظیم تراز نهایی دستگاه‌های بزرگ (مثل یخچال و لباسشویی).

۲. ابزارهای مونتاژ

تمام پیچ‌های حیاتی باید با **ترکمتر (Torque Wrench)** بسته شوند. چک کردن گشتاور بستن پیچ‌ها از وظایف مهم کنترل کیفی است.

  • **ترکمتر کالیبره:** برای بستن پیچ‌های موتور و وزنه‌های تعادل.
